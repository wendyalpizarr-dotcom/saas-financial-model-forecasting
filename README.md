# saas-financial-model-forecasting
Dynamic SaaS financial model and KPI dashboard built in Excel using subscription customer data.
Project Overview

Built a dynamic financial analytics and forecasting model for a subscription-based SaaS company using Excel and multiple customer and revenue datasets.

The project combines financial modeling, dashboard design, KPI analysis, scenario forecasting, and business intelligence techniques commonly used in FP&A and strategic finance roles.

Business Problem

Subscription businesses depend heavily on recurring revenue, customer retention, and efficient customer acquisition.

The objective of this model was to:

Analyze historical performance
Track SaaS KPIs
Forecast future revenue
Evaluate customer economics
Support strategic decision making
